Certificate of Proficiency in Information Systems Management.

A solid foundation to start your career in Information Technology, This semester will not only prepare you for your future studies at APTECH but will also provide the know how to better understand and use existing Computer Based Systems deployed in all modern day offices.

Students will learn the following modules in this semester:

Module

  • Office Automation
  • Programing Principles and Techniques
  • Logic Building and Elementary Programming
  • Building Next Generation Websites
  • UI/UX for Responsive Design
  • Data Management(SQL Server)
  • eProject-Website Development(HTML 5)

Workshops

  • Photoshop
  • WordPress
  • Joomla

Job Profile

  • Website Designer
  • HTML5 Programer
  • C Programmer
  • Database Developer

Learning Objectives

  1. Gain essential skills for producing professional work with the Microsoft Office suite including Word, Excel, PowerPoint, and Outlook.

  2. Learn generic programming principles

  3. Use programming constructs to write programs in C

  4. Use HTML5/CSS3/JavaScript to develop interactive Websites and apps

  5. Create user-friendly and responsive using UI/UX design principles

  6. Microsoft Visual Studio 2013 Professional Edition

  7. Normalize raw data into well-organized database tables in SQL Server

  8. Develop an interactive Website using HTML5
