Diploma in Information Systems Management (DISM)

Semester II builds on the knowledge of CPISM (Semester I) and focuses on designing full-fledged desktop applications using JAVA, while introducing the student to another object-oriented language C#. This semester also prepares the student for the Exam 70-536: Microsoft .NET Framework - Application Development Foundation. The curriculum contents are specially designed for those who are keen on getting themselves a career in the field of application development.

Students will learn the following modules in this semester:

Learning Objectives

  1. Use XML and JSON to store and exchange data.
  2. Develop object-oriented programming skills using OOP principles and concepts.
  3. Design and develop desktop applications using Java.
  4. Develop basic and advanced object-oriented applications using C#
  5. Learn database programming in Windows and includes extensive coverage of powerful .NET Windows Forms features, such as modern controls, data

    binding, and Click once deployment for creating smart client applications.
  6. Microsoft Visual Studio 2013 Professional Edition
  7. Move your applications stored on a computer to a remote location and make them accessible online through a standard browse.
  8. Develop a real-world application using .NET technologies.
